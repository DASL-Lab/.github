# DASL Lab: Data-driven Advancements in Statistical Learning

Current projects:

- Proportions of variants of concern in wastewater samples space and time
  - See [this notebook](https://github.com/DASL-Lab/lineage-proportion-estimators/blob/main/notebooks/wastewater-data-explanation.ipynb) for a description of the data.
  - Estimation of the proportions using GAM and Machine Learning methods.
    - For example, [Basic Linear Models](https://github.com/DASL-Lab/lineage-proportion-estimators/blob/main/notebooks/basic_modelling.ipynb) and [Time-Varying Coefficients Models](https://github.com/DASL-Lab/lineage-proportion-estimators/blob/main/notebooks/varying-coefficient-model.ipynb).
  - Estimation of the *variants* using unsupervised/clustering methods
  - [Analysis of residuals and model diagnostics](https://github.com/DASL-Lab/lineage-proportion-estimators/blob/main/notebooks/model_diagnostics.ipynb) for reconstructions of wastewater mutations data.
  - Determination of the "correct" number of wastewater sampling sites
    - See [this paper](https://www.sciencedirect.com/science/article/pii/S1755436525000222) for the basics.
- Modifications to DBSCAN based on the structure of the data, in particular a change in the definition of "central" points to include relations, rather than distances.
